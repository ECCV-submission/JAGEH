# JAGEH
Joint-modal Adaptive Graph-Enhancing Hashing for Unsupervised Cross-modal Retrieval

## Usage
### Requirements
- python == 2.7.x
- pytorch == 1.0.1
- torchvision
- CV2
- PIL
- h5py
